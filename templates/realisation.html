Projet : Mail Automatique

Description

J'ai développé une application qui permet de s'enregistrer via une fiche utilisateur en renseignant des informations personnelles. Ensuite, un système de tâche planifiée envoie un mail de manière automatique à l'employé le jour de son anniversaire et le jour de son anniversaire d'ancienneté au sein de l'entreprise.

L'employé a le choix de recevoir les mails ou non, de préciser quel mail il souhaite recevoir ou de recevoir les deux. Ce choix se fait lors du remplissage du formulaire.

Technologies utilisées

  • Windev
  • Webdev
  • Teams (Communications)

Conclusion

Ce projet m'a permis de mieux comprendre la gestion des tâches planifiées et la communication entre les équipes internationales. J'ai pu développer mes compétences en Windev et Webdev tout en travaillant sur un projet concret qui améliore la cohésion d'équipe au sein de l'entreprise.

Projet : Jeu d'aventure et de combat

Description

J'ai développé un mini jeu dans le terminal Python dans lequel il est possible de créer son personnage, choisir des coups spéciaux et des attaques. On commence le jeu avec un certain nombre de points de vie et le principe est de pouvoir se balader à travers le lycée, aller discuter avec des gens, aller à la cafétéria pour récupérer de la nourriture et aller au camp d'entraînement.

Au camp d'entraînement, on peut se battre contre des bots, on peut choisir d'attaquer, manger pour regagner des points de vie ou alors fuir le combat.

Technologies utilisées

  • Python
  • Visual Studio Code
  • GitHub

Conclusion

Ce projet m'a permis de développer mes compétences en Python et en gestion de projet. J'ai appris à structurer un code orienté objet, à gérer des interactions utilisateur et à créer une expérience de jeu cohérente malgré les limitations d'une interface en mode texte.

Projet : Ticket JO

Description

J'ai développé un système de création de tickets pour les Jeux Olympiques avec un QR code à scanner en fonction de la place et du prix. Cette application permet de générer des tickets personnalisés avec toutes les informations nécessaires pour assister à un événement sportif.

Le système intègre une gestion des places, des tarifs et la génération automatique de QR codes sécurisés pour l'accès aux différentes épreuves.

Technologies utilisées

  • Python
  • GitHub
  • Visual Studio Code

Conclusion

Ce projet m'a permis de renforcer mes compétences en Python et d'utiliser de nouvelles bibliothèques pour la génération de QR codes et la manipulation d'images. J'ai également appris à concevoir un système de billetterie simple mais fonctionnel.

Projet : API avec interface utilisateur

Description

J'ai développé avec 3 camarades un site internet utilisant des requêtes API pour obtenir des informations sur le championnat de la Ligue 1 sur la saison 2023. Le site comprend un système de connexion utilisateur et plusieurs fonctionnalités comme la possibilité d'ajouter des équipes en favoris, faire des recherches et bien d'autres.

L'application permet de consulter le classement, les résultats des matchs, les statistiques des joueurs et des équipes en temps réel grâce à l'API de la Ligue 1.

Technologies utilisées

  • Django (framework Python)
  • SQL
  • GitHub
  • Trello
  • Visual Studio Code

Conclusion

Ce projet m'a permis de travailler en équipe avec une gestion du temps et une organisation des tâches à réaliser pour chaque membre du groupe. J'ai approfondi mes connaissances en Django, en gestion de base de données et en intégration d'API.

Projet : Adaptation de feuilles de pesée Excel

Description

J'ai adapté des feuilles de pesée que le service production de l'entreprise utilise pour créer la diphotérine que nous vendons. J'ai dû utiliser Excel et notamment VBA pour mettre en place des macros afin de récupérer le poids d'un produit sur une balance via un bouton directement présent sur la feuille Excel.

J'ai également paramétré un scanner afin de lire les étiquettes des produits et afficher les bonnes données dans les cases correspondantes. Enfin, j'ai ajouté un raccourci clavier pour effectuer la même tâche que le bouton qui récupère le poids.

Technologies utilisées

  • Excel (VBA)
  • Teams

Conclusion

Ce projet m'a permis de travailler en situation réelle, à la demande du chef de production avec un délai à respecter. Cela m'a permis d'apprendre à m'organiser, à gérer mon temps et à prioriser les tâches à exécuter. J'ai également pu approfondir mes connaissances en VBA et en automatisation de processus.

Projet : Gestion Entretien

Description

J'ai développé un programme uniquement en backend en faisant des requêtes API sur une base de données. Le principe était de pouvoir faire des requêtes API pour récupérer les informations d'une table, ajouter une nouvelle ligne dans la table, supprimer une ligne et en modifier une.

Cette API REST complète permet de gérer efficacement les entretiens professionnels, avec toutes les fonctionnalités CRUD nécessaires.

Technologies utilisées

  • TypeScript
  • SQL
  • Postman
  • Visual Studio Code
  • GitHub

Conclusion

Ce projet m'a permis de m'améliorer dans la création et l'utilisation des requêtes API et de découvrir les bases du TypeScript. J'ai appris à structurer correctement une API REST et à tester efficacement ses fonctionnalités avec Postman.
