Projet : GestEPI

TypeScript React VSCode Ecole

Description du projet

Le projet GestEPI vise à créer une application pour gérer les Équipements de Protection Individuelle (EPI) utilisés par des cordistes dans une entreprise de travaux en hauteur.

L’objectif est de faciliter le suivi des EPI et de leurs contrôles grâce à une application web. Les fonctionnalités principales incluent la saisie des EPI, le suivi des contrôles, la consultation des historiques et la gestion d’alertes pour les vérifications à venir.

Le projet est structuré autour d’une architecture en React (front), Express + MariaDB (back) et un package partagé pour les interfaces de types TypeScript.

Conclusion

Ce projet permet de moderniser la gestion des EPI en remplaçant les fichiers papier ou tableurs par une solution numérique complète. Cela assure un meilleur suivi, plus de sécurité pour les utilisateurs, et une centralisation des données utile à l’entreprise.
