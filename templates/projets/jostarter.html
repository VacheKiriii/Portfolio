Projet : JoStarter

Django Javascript HTML École

Description du projet

Le projet JO Tickets consiste à créer un système complet de gestion de billets pour les matchs de football des Jeux Olympiques. Il repose sur une API développée avec Django, et comprend deux applications : l’une pour les supporters (achat de billets, génération de QR codes), l’autre pour les stadiers (scannage des billets). Les billets sont liés à une base de données centralisée, et la communication entre les applications se fait via des appels API sécurisés.

Conclusion

Ce projet propose une solution moderne et fonctionnelle pour gérer la billetterie d’un événement sportif, en assurant une expérience fluide pour les supporters et un contrôle rapide pour les stadiers, grâce à une API centralisée et des interfaces dédiées.
